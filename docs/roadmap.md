# Cowork 开发路线图

## 项目概览

**目标**：构建一个支持多 Worker 并行执行、可扩展布局的个人任务调度平台。

**技术栈**：
- 后端：Go + SQLite
- 前端：React + TypeScript + Zustand
- 通信：REST + WebSocket

---

## Phase 1：基础架构（预计 1-2 周）

### 目标

## 开发优先级原则

1. **先跑起来** — Phase 1 优先级最高
2. **核心功能优先** — 任务系统 > Dashboard > Agent
3. **迭代交付** — 每个 Phase 完成后可独立运行
4. **可扩展设计** — 为后续功能预留接口
